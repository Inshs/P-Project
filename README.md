# P-Project
중고차 AI 시세 분석 예측 서비스

## 시작하기 (Getting Started)

이 프로젝트는 Flutter로 개발되었습니다.

### 필수 조건 (Prerequisites)
- Flutter SDK
- Java Development Kit (JDK) 17 이상
- Visual Studio Build Tools (Windows 개발 시)

### 실행 방법 (How to Run)

1. **의존성 설치**
   ```bash
   flutter pub get
   ```

2. **앱 실행**
   - **Windows**:
     ```bash
     flutter run -d windows
     ```
   - **Chrome (Web)**:
     ```bash
     flutter run -d chrome
     ```
