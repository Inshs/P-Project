# 프로젝트 개발 보고서: 중고차 시세 예측 서비스

## 1. 프로젝트 개요
본 프로젝트는 **중고차 시세 예측 AI 서비스**의 프론트엔드 애플리케이션 개발을 목표로 합니다. Flutter 프레임워크를 사용하여 Windows 데스크톱 및 웹 환경에서 실행 가능한 크로스 플랫폼 앱을 구축하고 있습니다.

## 2. 개발 환경 구축 (Troubleshooting)
초기 개발 환경 설정 과정에서 발생한 주요 이슈와 해결 내역입니다.

### 2.1. 주요 이슈 및 해결
1.  **JDK (Java Development Kit) 누락**
    *   **증상**: Android 빌드 도구 요구사항 충족 실패.
    *   **해결**: `winget`을 통해 **Microsoft OpenJDK 17** 설치 완료.
2.  **Flutter SDK 손상**
    *   **증상**: `flutter doctor` 실행 시 파일 누락 오류 발생.
    *   **해결**: `git` 명령어로 손상된 파일 복구 및 캐시 초기화(`bin/cache` 삭제) 후 재빌드.
3.  **Visual Studio Build Tools 부재**
    *   **증상**: Windows 데스크톱 앱 빌드 실패 (`Unable to find suitable Visual Studio toolchain`).
    *   **해결**: **Visual Studio 2022 Build Tools** 설치 (C++ 데스크톱 개발 워크로드 및 Windows 10 SDK 포함).

## 3. 구현 기능 (Features)

### 3.1. 메인 홈 화면 (`lib/main.dart`)
*   **로그인 UI**: 아이디/비밀번호 입력, 소셜 로그인(네이버, 카카오, 구글) 버튼.
*   **차량 리스트**: '최근 조회 차량' 및 '인기 모델 추천'을 가로 스크롤 리스트로 제공.
*   **네비게이션 바**: 하단 탭바(내 차 찾기, 내 차 시세, 설정) 구현.
*   **비회원 접근**: "로그인 없이 시세 조회하기" 버튼을 통해 차량 정보 입력 화면으로 진입 가능.

### 3.2. 차량 정보 입력 화면 (`lib/car_info_input_page.dart`)
*   **기본 정보 입력**:
    *   브랜드/모델 선택 (Dropdown)
    *   연식 선택 (Dropdown)
    *   주행거리 입력 (TextField)
    *   연료 타입 선택 (ChoiceChip)
*   **상세 옵션 선택**:
    *   성능 점검 별점 (Star Rating)
    *   무사고 여부 및 주요 옵션(선루프, 내비게이션 등) 체크박스.
    *   지역 선택.
*   **화면 연결**: 메인 화면에서 네비게이션(`Navigator.push`)을 통해 이동하도록 구현.

### 3.3. 시세 예측 결과 화면 (`lib/result_page.dart`)
*   **구조**: 가격 분석, 구매 타이밍, AI 조언의 3개 탭으로 구성.
*   **가격 분석 탭**:
    *   신뢰도 원형 그래프.
    *   비슷한 차량 가격 분포 막대 그래프 (`fl_chart`).
*   **구매 타이밍 탭**:
    *   구매 적기 신호등 UI.
    *   거시경제, 트렌드, 신차 일정 타이밍 지표 (Circular Indicators).
*   **AI 조언 탭**:
    *   AI 상세 조언 텍스트.
    *   허위매물 위험도 지표.
    *   문자 복사 및 전화 대본 기능 버튼 -> `NegotiationPage`로 이동.

### 3.4. 네고 도우미 화면 (`lib/negotiation_page.dart`)
*   **기능**: 사용자가 판매자와 협상할 때 유용한 도구 제공.
*   **구성요소**:
    *   **협상 포인트 점검**: 타이어, 가격 등 체크리스트.
    *   **문자 전송 모드**: AI가 작성한 정중한 협상 메시지 제공 및 복사 기능.
    *   **전화 통화 모드**: 단계별(인사-네고-마무리) 통화 대본 및 팁 제공.

### 3.5. 마이페이지 (`lib/mypage.dart`)
*   **구조**: 찜한 차량 / 최근 분석 탭으로 구성.
*   **기능**:
    *   **찜한 차량**: 사용자가 좋아요(하트)를 누른 차량 목록 표시.
    *   **최근 분석**: 최근 조회한 차량 목록 표시. 좋아요/알림 아이콘 클릭 시 상태 변경 및 리스트 반영.
    *   **상호작용**: 좋아요 클릭 시 '찜한 차량' 탭에 즉시 반영, 알림 클릭 시 토글 상태 변경 (기능 준비중).

### 3.6. 설정 화면 (`lib/settings_page.dart`)
*   **기능**: 앱 환경 설정 관리.
*   **구성요소**:
    *   **다크 모드**: 앱 전체 테마(Light/Dark) 전환 스위치.
    *   **AI 엔진**: API 키 설정 (UI만 구현).
    *   **지원 및 정보**: 기록 삭제, 문의하기, 버전 정보.

### 3.7. 메인 네비게이션 및 테마 (`lib/main.dart`)
*   **구조**: `IndexedStack`을 사용하여 하단 탭 전환 시 화면 상태 유지.
*   **테마 관리**: `ThemeMode` 상태를 최상위 위젯에서 관리하여 다크 모드 전환 시 앱 전체에 즉시 반영.
*   **탭 구성**: 홈, 내 차 찾기, 마이페이지, 설정.

## 4. 기술 스택
*   **Framework**: Flutter (Dart)

*   **Libraries**:

*   **Design**: Material Design 3 (Custom Theming)
